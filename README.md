# Weather App- HTML, CSS & JAVASCRIPT

Web application that allows users to check the current weather from any city using the OpenWeather API.
this application was built with **HTML, CSS & Vanilla Javascript**, featuring dynamic themes, persisted search history and
an animated UI.
Developed as part of a portfolio project for Frontend + API integration practice.

---

## Technologies Used

| Technology | Purpose |
|-----------|--------|
| HTML      | Structure |
| CSS       | Styling + animations |
| JavaScript (Vanilla) | Logic & API handling |
| OpenWeather API | Weather data |
| LocalStorage | Save theme & last city |

## Installation & local Setup


1. **clone the repository**
 ```
     Git clone https://github.com/adalbvp/weather-app.git
```
2. **Create a config.js file in the root folder**
   ```
     window.CONFIG = {
       API_KEY: "YOUR_OPENWEATHER_API_KEY_HERE"
     };
   ```
3. **Run the project**
   ```
     open index.html directly in the browser OR
